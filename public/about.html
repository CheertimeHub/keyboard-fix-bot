Xevra.GPT คืออะไร?

Xevra.GPT คือ Discord bot และเว็บแอปสำหรับแก้ข้อความที่พิมพ์ผิด keyboard layout เมื่อคุณพิมพ์ภาษาไทยแต่ keyboard ยังอยู่ในโหมดอังกฤษ หรือในทางกลับกัน หรือแม้แต่ตอนที่ Caps Lock ติดอยู่โดยไม่รู้ตัว Xevra.GPT จะจัดการให้อัตโนมัติ

รองรับ 3 กรณี

ไทย → อังกฤษ อังกฤษ → ไทย Caps Lock ติด

ใช้งานบนเว็บได้เลยที่หน้า Home หรือจะเชิญ Xevra.GPT เข้า Discord server แล้ว reply ข้อความที่พิมพ์ผิดแล้ว @Xevra.GPT ก็ได้เหมือนกัน
