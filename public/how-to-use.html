🎲 ทอยเต๋า

ทอยเต๋าแบบกำหนดเองได้ รองรับ dice notation มาตรฐาน (NdM) และภาษาพูดธรรมชาติ

ตัวอย่าง
User @Xevra.GPT ทอยเต๋า
Xevra 🎲 ทอย d6: 4
User @Xevra.GPT ทอย 2d6
Xevra 🎲 ทอย 2d6: 9
User @Xevra.GPT ทอย 3#d6
Xevra 🎲 ทอย 3d6: [2, 5, 4] = 11
User @Xevra.GPT ทอย 3 ลูก 6 หน้า
Xevra 🎲 ทอย 3d6: [1, 6, 3] = 10
NdM = ทอย N ลูก M หน้า ได้ผลรวม
N#dM = ทอย N ลูก M หน้า ได้ผลแยกรายลูก

🔢 สุ่มเลข

สุ่มตัวเลขในช่วงที่กำหนด พิมพ์แบบไหนก็ได้ตามสบาย

ตัวอย่าง

Xevra 🔢 สุ่ม 1-100: 42
User @Xevra.GPT ช่วยสุ่มเลข 1 ถึง 50 ให้หน่อย
Xevra 🔢 สุ่ม 1-50: 27
User @Xevra.GPT สุ่มเลข 10
Xevra 🔢 สุ่ม 1-10: 7

✨ เลือกให้

ให้ Xevra.GPT ช่วยเลือกเมื่อตัดสินใจไม่ได้ คั่นตัวเลือกด้วย "หรือ" หรือ ","

ตัวอย่าง
User @Xevra.GPT เลือกให้หน่อย ข้าวผัด หรือ ราดหน้า
Xevra ✨ เลือก ราดหน้า (จาก ข้าวผัด, ราดหน้า)
User @Xevra.GPT ช่วยเลือกระหว่าง A, B, C ให้หน่อย
Xevra ✨ เลือก B (จาก A, B, C)
ใช้ "หรือ" / "กับ" / "," ในการคั่นตัวเลือก — ใส่กี่ตัวเลือกก็ได้
