🎲 ทอยเต๋า

รองรับ dice notation มาตรฐาน สามารถใส่ modifier, keep/drop, และ threshold ได้

พื้นฐาน
User @Xevra.GPT ทอยเต๋า
Xevra 🎲 4 (d6)
User @Xevra.GPT ทอย 2d6
Xevra 🎲 [5, 4] = 9 (2d6)
Modifier (+/-)
User @Xevra.GPT 5#d20+5
Xevra 🎲 5#d20+5
17 12 8 21 24
Keep Highest / Keep Lowest
User @Xevra.GPT 4d6kh3 (stat generation)
Xevra 🎲 [6, 5, 4, 1] = 15 (4d6kh3)
User @Xevra.GPT 10d6kl2
Xevra 🎲 [6, 5, 5, 4, 4, 3, 2, 2, 1, 1] = 2 (10d6kl2)
Drop Lowest / Drop Highest
User @Xevra.GPT 4d6dl1 (= 4d6kh3)
Xevra 🎲 [6, 5, 4, 1] = 15 (4d6dl1)

User @Xevra.GPT 5#d20<5
Xevra 🎲 5#d20<5

Successes: 1/5

Xevra 🎲 attack: [5, 3] + 3 = 11 (2d6+3)
User @Xevra.GPT ทอย โจมตี d20
Xevra 🎲 โจมตี: 17 (d20)
Syntax: [N#][M]dS[+/-mod][kh/kl/dh/dl N][</>/<=/>= N]

N# = ทอยซ้ำ N ครั้ง  |  MdS = M ลูก S หน้า
kh/kl = keep highest/lowest  |  dh/dl = drop highest/lowest
<5 >10 >=15 = threshold สำเร็จ/ล้มเหลว

💬 ใส่ "ช่วย" / "ขอ" นำหน้า หรือ "ให้หน่อย" / "ครับ" ท้ายได้เลย

(จาก A, B, C)
